# UI/UX 요구사항 상세

## 🎨 디자인 시스템

### 색상 팔레트
```css
Primary: #3B82F6 (Blue)
Secondary: #10B981 (Green)
Danger: #EF4444 (Red)
Warning: #F59E0B (Orange)
Background: #F9FAFB (Light Gray)
Text: #111827 (Dark Gray)
```

### 타이포그래피
- 헤딩: Bold, 24px-32px
- 본문: Regular, 16px
- 작은 텍스트: Regular, 14px

### 간격 시스템
- 4px 단위 (4, 8, 12, 16, 24, 32, 48, 64)

## 📱 페이지별 상세 요구사항

### 1. HomePage

#### 레이아웃
```
┌─────────────────────────────────┐
│  Header (알림 설정, 위치 변경)    │
├─────────────────────────────────┤
│  현재 위치: 서울시 강남구         │
│  [위치 변경 버튼]                │
├─────────────────────────────────┤
│  날씨 정보 카드                  │
│  🌤️ 15°C, 맑음                  │
│  습도 60%, 풍속 10km/h           │
├─────────────────────────────────┤
│  미세먼지 정보 카드              │
│  🟢 좋음 (PM10: 17, PM2.5: 6)   │
├─────────────────────────────────┤
│  다음 알림: 오전 8시 (2시간 후)  │
│  [알림 설정 버튼]                │
└─────────────────────────────────┘
```

#### 기능
- 실시간 날씨 정보 표시
- 실시간 미세먼지 정보 표시
- 다음 알림 시간 표시
- 빠른 액션 버튼

### 2. 사용자 위치 설정 페이지

#### 레이아웃
```
┌─────────────────────────────────┐
│  위치 설정                       │
├─────────────────────────────────┤
│  [현재 위치 사용] 버튼           │
├─────────────────────────────────┤
│  주소 검색                       │
│  [검색 입력창]                   │
│  검색 결과 목록                  │
├─────────────────────────────────┤
│  또는                            │
├─────────────────────────────────┤
│  지도에서 선택                   │
│  [지도 영역]                     │
└─────────────────────────────────┘
```

#### 기능
- 현재 위치 자동 감지
- 주소 검색 (카카오/네이버 지도 API)
- 지도에서 위치 선택
- 저장 버튼

### 3. AlertSettingsPage 개선

#### 레이아웃
```
┌─────────────────────────────────┐
│  알림 설정                       │
├─────────────────────────────────┤
│  새 알림 만들기                  │
│  ┌─────────────────────────┐   │
│  │ 알림 이름: [입력]       │   │
│  │ 시간: [08:00]           │   │
│  │ 요일: [월][화][수]...   │   │
│  │ 알림 타입:              │   │
│  │ ☑ 날씨  ☑ 미세먼지      │   │
│  │ ☐ 버스  ☐ 지하철        │   │
│  │ [저장]                  │   │
│  └─────────────────────────┘   │
├─────────────────────────────────┤
│  내 알림 (3개)                   │
│  ┌─────────────────────────┐   │
│  │ 출근 알림        [ON]   │   │
│  │ 매일 08:00              │   │
│  │ 날씨, 미세먼지          │   │
│  │ [수정] [삭제]           │   │
│  └─────────────────────────┘   │
└─────────────────────────────────┘
```

#### 개선 사항
- 시간 선택 UI (Cron 대신)
- 요일 선택 UI
- 알림 타입 아이콘
- 카드 형태 알림 목록
- 활성화/비활성화 토글

## 🎯 사용자 경험 목표

1. **직관적**: 3번의 클릭으로 알림 설정 가능
2. **빠른**: 로딩 시간 최소화
3. **명확한**: 에러 메시지와 피드백 명확
4. **접근성**: 키보드 네비게이션, 스크린 리더 지원
