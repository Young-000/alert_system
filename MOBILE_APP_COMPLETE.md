## 📱 주요 변경사항

### 레이아웃
- 데스크톱: `max-w-7xl` → 모바일: `max-w-md`
- 하단 네비게이션 바 추가 (홈, 알림, 위치)
- 헤더 높이 감소 (h-16 → h-14)

### 컴포넌트
- `Card` → `MobileCard` (모든 페이지)
- 버튼 크기 조정 (sm, md, lg)
- 입력 필드 모바일 최적화

### 스타일
- 둥근 모서리 증가 (rounded-lg → rounded-xl, rounded-2xl)
- 간격 조정 (모바일 친화적)
- 색상 대비 개선
