## 기능 상세

### Backend: `GET /notifications/stats`
- TypeORM QueryBuilder로 `GROUP BY status` 집계
- `?days=N` 쿼리 파라미터로 기간 필터링
- `successRate = (success / total * 100)` (소수점 1자리)
- 빈 데이터: `{ total: 0, successRate: 100 }`

### Frontend: `NotificationStats` 컴포넌트
- 전체 건수, 성공률, 실패 건수 표시
- 색상 코딩: 녹색(>90%), 노란색(>70%), 빨간색(<=70%)
- 수평 상태 바 + 텍스트 범례 (색맹 대응)
- 스켈레톤 로딩 (CSS 클래스)
- total=0이면 숨김
