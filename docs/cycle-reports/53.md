## Technical Decisions
- 외부 ML 라이브러리 미사용 → 번들 사이즈 증가 없음
- Bayesian estimator로 소량 데이터에서도 합리적 예측 (prior shrinkage)
- zero-variance 컬럼 자동 감지하여 singular matrix 방지
- @Optional() + @Inject() 패턴으로 서비스 미등록 시 graceful degradation
- 기존 use-case에 ML 위임 추가하되 모든 기존 테스트 통과 보장

## Architecture

```
CommuteRecord → FeatureEngineering → FeatureRow[]
                                          ↓
                    EnhancedPatternAnalysis (day/weather/seasonal/route)
                                          ↓
                              UserPattern (persisted)
                                          ↓
                         PredictionEngine (5-tier Bayesian)
                                          ↓
                    PredictionResult { time, confidence, factors, tier }
```
