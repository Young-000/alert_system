# Store Screenshots Guide

## Required Screenshots

### iOS (App Store Connect)
| Device | Size | Count |
|--------|------|-------|
| iPhone 6.7" (15 Pro Max) | 1290 x 2796 | 3-10 |
| iPhone 6.5" (14 Plus) | 1284 x 2778 | 3-10 |
| iPad 12.9" (optional) | 2048 x 2732 | 3-10 |

### Android (Google Play Console)
| Type | Size | Count |
|------|------|-------|
| Phone | 1080 x 1920 (min) | 2-8 |
| Tablet 7" (optional) | 1200 x 1920 | 2-8 |
| Tablet 10" (optional) | 1920 x 1200 | 2-8 |

## Recommended Screenshot Sequence

1. **홈 화면** — 출근 브리핑 (날씨 + 미세먼지 + 교통)
2. **알림 설정** — 맞춤 알림 목록 + 토글
3. **위젯** — iOS/Android 홈 화면 위젯
4. **경로 설정** — 경로 관리 화면
5. **알림 기록** — 지난 알림 히스토리

## Screenshot Generation

Screenshots will be captured during TestFlight/Internal Testing phase.
Use Expo's `npx expo run:ios` and `npx expo run:android` for device captures.
