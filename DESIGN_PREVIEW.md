# 🎨 사이트 디자인 미리보기

## 접속 방법

### 로컬 개발 서버
```bash
# 프론트엔드 (터미널 1)
cd frontend
npm run dev

# 백엔드 (터미널 2)
cd backend
npm run start:dev
```

### 접속 URL
- **프론트엔드**: http://localhost:5173
- **백엔드 API**: http://localhost:3000

## 📱 모바일 디자인 특징

### 1. 홈 페이지
- **헤더**: 간소화된 헤더 (로고 + 로그아웃)
- **인사말**: "안녕하세요, [이름]님 👋"
- **위치 정보**: 주소 표시 또는 위치 설정 버튼
- **날씨 카드**: 
  - 큰 이모지 아이콘
  - 온도 (큰 글씨)
  - 날씨 상태
  - 습도/풍속 정보
- **미세먼지 카드**:
  - 상태 이모지 (🟢/🟡/🔴)
  - 상태 텍스트
  - PM10, PM2.5, AQI 수치
- **하단 네비게이션**: 홈, 알림, 위치

### 2. 알림 설정 페이지
- **푸시 알림 활성화 배너**: 노란색 배경 (권한 요청)
- **새 알림 만들기 카드**:
  - 알림 이름 입력
  - 시간 선택 (시:분 드롭다운)
  - 요일 선택 (7개 버튼 그리드)
  - 알림 타입 선택 (4개 아이콘 버튼)
- **내 알림 목록**:
  - 각 알림 카드
  - 활성/비활성 배지
  - 시간 표시
  - 알림 타입 태그
  - 삭제 버튼

### 3. 위치 설정 페이지
- **현재 위치 사용 버튼**: 큰 버튼
- **주소 입력 필드**
- **위도/경도 입력** (2열 그리드)
- **저장/취소 버튼**

### 4. 로그인 페이지
- **중앙 정렬 카드**
- **로고**: "Alert" 큰 글씨
- **이메일/이름 입력**
- **시작하기 버튼**

## 🎨 디자인 시스템

### 색상
- **Primary**: #3B82F6 (파란색)
- **Secondary**: #10B981 (초록색)
- **Danger**: #EF4444 (빨간색)
- **Background**: #F9FAFB (연한 회색)

### 타이포그래피
- **제목**: 2xl (24px), bold
- **부제목**: lg (18px), semibold
- **본문**: base (16px), regular
- **작은 텍스트**: sm (14px), xs (12px)

### 간격
- **카드 간격**: 12px (space-y-3)
- **내부 여백**: 16px (p-4)
- **하단 네비게이션 높이**: 64px

### 둥근 모서리
- **카드**: rounded-2xl (16px)
- **버튼**: rounded-xl (12px)
- **작은 요소**: rounded-lg (8px)

## 📐 레이아웃

### 최대 너비
- **모바일**: max-w-md (448px)
- **중앙 정렬**: mx-auto
- **좌우 여백**: px-4 (16px)

### 하단 네비게이션
- **고정 위치**: fixed bottom-0
- **배경**: 흰색 + 상단 테두리
- **아이템**: 3개 (홈, 알림, 위치)
- **활성 상태**: 파란색 텍스트

## 🎯 인터랙션

### 버튼
- **기본**: 파란색 배경, 흰색 텍스트
- **호버**: 더 진한 파란색
- **활성**: scale-98 (약간 축소)
- **로딩**: 스피너 애니메이션

### 카드
- **기본**: 흰색 배경, 그림자
- **터치 피드백**: active:scale-98
- **스와이프**: 좌우 스와이프 지원

### 입력 필드
- **포커스**: 파란색 테두리 + 그림자
- **에러**: 빨간색 테두리 + 에러 메시지

## 📱 모바일 최적화

### 터치 영역
- **최소 크기**: 44x44px
- **버튼**: 충분한 패딩

### 스크롤
- **부드러운 스크롤**: -webkit-overflow-scrolling: touch
- **하단 여백**: pb-20 (하단 네비게이션 공간)

### Safe Area
- **상단**: env(safe-area-inset-top)
- **하단**: env(safe-area-inset-bottom)

## 💡 디자인 하이라이트

1. **깔끔한 카드 디자인**: 정보를 카드로 구분
2. **직관적인 아이콘**: 이모지로 시각적 표현
3. **명확한 계층 구조**: 제목 → 내용 → 액션
4. **부드러운 애니메이션**: 터치 피드백, 전환 효과
5. **일관된 색상**: 파란색 계열로 통일
